# AI Ethics Analyzer

AI Ethics Analyzer is a web-based platform that evaluates the ethical risks of AI systems before they are developed.

The system analyzes AI ideas and identifies potential risks such as bias, privacy issues, transparency problems, and misuse.

## Features

- AI bias risk detection
- Privacy risk evaluation
- Transparency analysis
- Misuse risk identification
- AI compliance score generation
- Ethical improvement suggestions
- Step-by-step risk resolution
- Automated AI ethics report

## Technologies Used

Frontend:
- HTML
- Tailwind CSS
- JavaScript

Backend:
- FastAPI
- SQLAlchemy
- PostgreSQL

## How to Run the Project

1. Install dependencies

```
pip install -r requirements.txt
```

2. Start FastAPI server

```
uvicorn main:app --reload
```

3. Open the frontend

Open `index.html` or `audit.html` in your browser.

## Project Purpose

The goal of this project is to promote responsible AI development by helping developers identify ethical risks early in the AI design process.
